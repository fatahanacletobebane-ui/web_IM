Junção de Letras

Aprenda como as letras se conectam para formar palavras

Conexão Inicial

بـ + ـا = با
Ba + Alif = Ba

Conexão Média

ـبـ + ـيـ = ـبيـ
Letras no meio da palavra

Conexão Final

ـب + ـر = ـبر
Letras no fim da palavra

Palavra Completa

ب + ا + ب = باب
Bāb (Porta)
